# Agent A - DB Prompt (ANCLORA-FCCC-001)

Objective:
- Validate whether Supabase migration is required.
- For FCCC v1, confirm migration skip or define additive migration only if mandatory.

Checks:
1) Schema readiness for KPI aggregation from existing tables.
2) Migration decision documented with explicit rationale.
3) Rollback safety and verification checks documented.

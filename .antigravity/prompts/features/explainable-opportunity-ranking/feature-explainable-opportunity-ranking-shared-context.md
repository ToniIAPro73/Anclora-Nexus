# Shared Context - Explainable Opportunity Ranking

## Problema
El equipo comercial tiene scores de match, pero no una priorizacion explicable y accionable para decidir en que oportunidades invertir tiempo primero.

## Objetivo v1
- Unificar ranking de oportunidades con explicabilidad y accion recomendada.
- Reducir tiempo de priorizacion manual diaria.

## Entidades base
- `property_buyer_matches`
- `prospected_properties`
- `buyer_profiles`

# Dominio: Buyer Matching y Cierre Comercial

## Objetivo
Asegurar lenguaje claro para match score, estado comercial y comision estimada.

## Revisar
- Tabla/panel de matches.
- Mensajes de score y explicabilidad.
- Estados de pipeline comercial y activity log.

## Riesgos frecuentes
- "Sin titulo" y labels no accionables.
- Score sin contexto para agentes.
- Estado del match ambiguo entre vistas.

## Criterios de aprobacion
- Score entendible y consistente.
- Estado comercial equivalente en todas las pantallas.
- Campos de cierre (comision, comprador potencial) sin contradicciones.
